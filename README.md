# 🌋 Bajo el volcán ediciones - CMS & Web

## 📖 Sobre el Proyecto
Refactorización y migración del sitio web de la editorial literaria "Bajo el volcán". El proyecto evoluciona de una arquitectura estática (Astro + JSON) a una plataforma dinámica y escalable construida con el framework Django y base de datos PostgreSQL, conservando la interfaz visual original.

Este sistema incluye un panel de administración (CMS) a medida que permite al cliente final gestionar de forma autónoma:
* 📚 Catálogo de Libros y publicaciones.
* ✍️ Perfiles de Autores.
* 📰 Sección de Noticias y actualizaciones.

## 🛠️ Stack Tecnológico
* **Backend:** Python, Django
* **Base de Datos:** PostgreSQL
* **Frontend:** HTML5, CSS3, Bootstrap (Reutilizado del proyecto original)
* **Despliegue Planificado:** Render/Railway (Backend) + Hostinger (DNS)
